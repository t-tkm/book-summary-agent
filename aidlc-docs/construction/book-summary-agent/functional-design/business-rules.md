# Business Rules — book-summary-agent

## BR-01: ISBN バリデーション
- 正規化後のISBNは10桁または13桁でなければならない
- 違反時: エラーメッセージを stderr に出力してプロセスを終了する（exit code 1）

## BR-02: モデルデフォルト値
- `--model` 未指定時のデフォルトは `claude-sonnet-4-20250514`

## BR-03: JSON抽出優先順位
1. ` ```json ``` ` ブロック
2. ` ``` ``` ` ブロック
3. `{` 始まりのテキスト
4. 上記すべて失敗 → エラー

## BR-04: 出力フィールドの欠損処理
- Claudeが返したJSONに不明フィールドがある場合は空文字列をデフォルトとする
- タイトルが空でも出力は継続する（エラーにしない）

## BR-05: cover_url の表示条件
- テキスト出力時、`cover_url` が空文字列の場合はその行を省略する

## BR-06: エラー伝播
- claude CLI の終了コードが非0の場合は stderr をそのままエラーメッセージとして上位に伝播する
- stdout が UTF-8 でない場合もエラーとして伝播する
