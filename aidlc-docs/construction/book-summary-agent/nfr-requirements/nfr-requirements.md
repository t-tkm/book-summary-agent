# NFR Requirements — book-summary-agent

## パフォーマンス
- **応答時間**: Claude CLIのWeb検索を含むため10〜60秒を許容する（ユーザー向けCLIツール）
- **スループット**: 単一ユーザーのシーケンシャル実行を想定。並列処理は対象外

## 可用性
- **前提条件**: Claude Code CLIが利用可能で認証済みであること
- **ネットワーク**: インターネット接続が必要（WebSearch/WebFetch使用）
- **オフライン**: 動作しない（設計上の制約）

## セキュリティ
- **APIキー**: コードにハードコードしない。Claude Code CLIの認証機構に委譲する
- **入力サニタイズ**: ISBNはハイフン除去・ASCII数字のみフィルタするため、コマンドインジェクションリスクは低い
- **プロセス実行**: `tokio::process::Command` でのCLI呼び出しは引数配列渡しのためシェルインジェクション不可

## 信頼性
- **エラーハンドリング**: 全エラーは `anyhow` で伝播し、最終的に `main()` でユーザー向けメッセージとして表示する
- **部分的失敗**: JSON解析失敗時はエラーメッセージを表示して終了。サイレント失敗はしない

## 保守性
- **単一ファイル構成**: `src/main.rs` に全ロジックを集約。小規模ツールとして適切
- **依存関係の最小化**: reqwest削除済み。外部HTTP依存なし
- **ログ**: 進捗を `eprintln!` で stderr に出力。stdout はデータ出力専用

## 互換性
- **OS**: macOS / Linux（`claude` コマンドが存在する環境）
- **Rust**: 1.75以上
- **Claude Code CLI**: 2.x系
